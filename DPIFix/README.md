# 使用方法

1. 将文件 `DPIFix.cmd`，放在一个非外接硬盘的某个路径下（推荐C盘文档，记得不要删除）
2. 打开“任务计划程序”，新建基本任务。
   1. 起个名字（比如DPIFix，说明随意），
   2. 计划周期为“每次登陆时”，
   3. 计划目标为“启动程序”，
   4. 路径选择那个 DPIFix.cmd 的路径，
   5. 然后完成。
3. 重启(非关机再开机)两次。
